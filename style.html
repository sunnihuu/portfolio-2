Design System Rules

Typography

  • Use Work Sans for all UI, navigation, and body text.
  • Use Instrument Serif Italic sparingly for section intros and emphasis only.
  • Maintain consistent vertical rhythm and line heights.
  • Always use uppercase + letter-spacing for labels and metadata.

Colors

  • Dark background: Use --bg (#1E1C19)
  • Panels and sections: Use --surface (#2A2723)
  • Project cards: Use --card (#EDE7DA) with --text-dark text
  • Accents and hover states: Use --accent (#D0B76D)
  • Divider lines: Use --line (rgba(255,255,255,0.12))

Avoid

  • Drop shadows (use borders and whitespace instead)
  • Glassmorphism effects
  • Gradients
  • Rounded buttons or heavy UI elements
  • Too many accent colors
